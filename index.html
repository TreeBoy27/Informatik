Wie Bilder auf einem Computer formatiert werden

Ein Bild auf einem Computer kann als eine zweidimensionale Matrix von Pixeln betrachtet werden. Jeder Pixel wird durch einen binären Wert repräsentiert: 0 oder 1. In monochromen Bildern steht 0 typischerweise für Schwarz und 1 für Weiß. Diese Matrix kann als Tabelle visualisiert werden, wobei jede Zelle der Tabelle einem Pixel des Bildes entspricht.
